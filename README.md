# Portifolio
Meu portifólio pessoal.

Este repositório contém um site estático com seções de projetos, experiência profissional e um laboratório de segurança interativo. Uma barra de navegação fixa facilita o acesso rápido a cada parte do portfólio. O conteúdo é traduzido automaticamente para inglês ou português com base nas configurações do navegador e o laboratório permite reiniciar as simulações.
